Background

Chris Trevino works in the Information Technology field. In order to help him better assist people with their network issues, he proposed an idea of an application that would allow him to scan, identify, and monitor devices on a network.

Scope

The project will address the ability for a user on a web application to scan and retrieve details from multiple devices on a network. Multiple other features will try to be appended into the program too, such as a in-app contact list, the ability to favorite devices, and the ability to manually add devices.

Proposed Solution

In order to accomplish this, we have multiple tools that can be used. For the front-end user interface, we will be using React.JS, as well as the web coding elements that go with it, such as HTML, CSS, and Bootstrap for formatting. For the back-end, we will be using Django while utilizing Python to program the network- based functions the application will do. The databasing will be done via Microsoft SQL Server.

Highlights

This project will be unique in helping our client have a lightweight and easy way to perform network monitoring. Unlike SolarWinds, the Device Discovery Tool will be easier to use and friendlier to the user. It will also the user to access the functions via mobile web browser, for when they are not near a computer.
